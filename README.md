## **üì• Getting Started**

### **Requirements**
- **Node.js 20+** or higher  
- **A supported wallet** (Trust Wallet, MetaMask, Ledger, etc.)  
- **HTTPS** for **secure transactions**

### **Installation**
```bash
git clone https://github.com/your-username/flowfixer.git
cd flowfixer
npm install
```

### **Run the App**
```bash
npm start
```
Open **`https://localhost:3000`** in your browser.
